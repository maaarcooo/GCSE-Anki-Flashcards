# GCSE-Anki-Flashcards

- Uses pipe for Anki flashcards import
- The text format: Question | Answer

## Anki import guidelines

Import text file
- Use Anki desktop app
- Import the text file
- Select separate with **pipe**

Import apkg file
- Use Anki desktop app
- Import the apkg file

Select FSRS (Free Spaced Repetition Scheduler) in Anki App option
- An advanced algorithm
- Optimize the scheduling of flashcard reviews based on your memory patterns
- Improving retention compared to the older SM-2 algorithm
- Allows for more personalized and efficient learning by adjusting intervals based on how well you remember the material.
